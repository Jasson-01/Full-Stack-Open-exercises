# Full Stack Open - Exercises

Repositorio con mis soluciones y ejercicios realizados durante el curso **Full Stack Open**.

## 📚 Acerca del curso

**Full Stack Open** es un curso gratuito enfocado en el desarrollo web moderno con tecnologías como:

* JavaScript
* React
* Node.js
* Express
* MongoDB
* GraphQL
* TypeScript
* Testing
* CI/CD
* Docker

🔗 Curso oficial: https://fullstackopen.com/es/
